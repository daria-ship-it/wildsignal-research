# Wild Signal GEO Research Library

A static research library for [Wild Signal Agency](https://wildsignal.agency) — an AI intelligence firm helping brands navigate the new information economy through Generative Engine Optimization (GEO).

## Live Site

[https://daria-ship-it.github.io/wildsignal-research/](https://daria-ship-it.github.io/wildsignal-research/)

## About

This library contains 40 research articles and field guides on:
- Generative Engine Optimization (GEO) and AI search visibility
- How LLMs find, evaluate, and cite content
- Brand strategy for the citation economy
- Measurement frameworks for AI search
- PR and earned media in an AI-first world

## AI Discovery Files

- [`/llms.txt`](https://daria-ship-it.github.io/wildsignal-research/llms.txt) — Standard llms.txt index
- [`/llms-full.txt`](https://daria-ship-it.github.io/wildsignal-research/llms-full.txt) — Full content export for LLM ingestion
- [`/.well-known/llm-index.llmfeed.json`](https://daria-ship-it.github.io/wildsignal-research/.well-known/llm-index.llmfeed.json) — LLMFeed JSON index
- [`/sitemap.xml`](https://daria-ship-it.github.io/wildsignal-research/sitemap.xml) — XML sitemap

## Source

All articles originally published at [wildsignal.agency](https://wildsignal.agency). Each article page includes a canonical link pointing to the original source.
